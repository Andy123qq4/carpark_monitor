### Project Progress

Our team is developing an **AI-powered car park monitoring system** for China Hong Kong City's loading bay. The system uses computer vision to automatically recognize vehicle license plates, track parking duration, and alert security staff in real time — reducing the need for dedicated 24/7 manual monitoring.

**What's been built so far:**
- License plate detection pipeline using YOLO v9 + ONNX (tested on GF15–GF18 CCTV footage provided by CHKC)
- Plate validation engine for Hong Kong plate formats
- Temporal tracking and deduplication for accurate vehicle session logging
- Web dashboard for viewing detections and plate sessions

Prof. Leung provided valuable guidance on framing the solution as a **business workflow** — designing a phased approach where AI progressively reduces human effort, rather than attempting full automation from day one. I am now working with the team to develop this workflow alongside the Sino colleagues.
